# Biais Géo-Culturel IA - Dashboard

Ce projet contient un tableau de bord interactif développé avec **Streamlit** pour analyser les biais géographiques et culturels des modèles d'intelligence artificielle (LLMs).

## 📊 Fonctionnalités

- **Analyse des Connaissances Culturelles** : Cartographie de la précision des réponses sur le patrimoine français.
- **Biais de Centralisation** : Mesure de la tendance des modèles à privilégier Paris.
- **Performance & Écologie** : Comparaison des temps de réponse et de l'empreinte carbone estimée.

## 🚀 Installation et Lancement

1.  **Prérequis** : Avoir Python installé sur votr machine.
2.  **Lancement rapide** :
    *   Double-cliquez sur le fichier `lancer_app.bat`.
    *   Cela installera automatiquement les dépendances et lancera le dashboard dans votre navigateur.

## 📂 Structure du projet

*   `dashboard/` : Contient le code source de l'application (`app.py`).
*   `data/` (ou racine) : Contient les fichiers CSV de données (Benchmark, Tests).
*   `lancer_app.bat` : Script d'automatisation pour Windows.

## 🛠️ Technologies

*   Python
*   Streamlit
*   Plotly (Visualisation)
*   Pandas (Analyse de données)
